Depress

Software to automatize creation of djvu files.

How to use

  • Compile software
  • Install djvulibre
  • Copy depress.exe to djvulibre folder (i.e. "c:\Program Files (x86)\DjVuLibre\")
  • Run depress.exe pictures_list.txt output.djvu
